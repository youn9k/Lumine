# 로컬 미디어 플레이어 앱 기획서

## 프로젝트 개요
- **프로젝트명**: (추후 결정)
- **개발 기간**: 1~2개월
- **타겟 플랫폼**: iPadOS, macOS (visionOS는 추후 대응)
- **목적**: 로컬 비디오 파일을 간편하게 재생하는 플레이어 앱

## 핵심 기능

### 1. 듀얼 UI 모드
- **일반 모드**: 전통적인 비디오 플레이어 UI
  - 가로/세로 방향 모두 지원
  - 재생 컨트롤러가 하단에 위치
  
- **숏폼 모드**: TikTok/YouTube Shorts 스타일
  - 세로 방향 전체화면 재생
  - 위/아래 스와이프로 이전/다음 영상 이동
  - 탭하여 재생/일시정지
  - 화면 비율이 맞지 않는 영상은 레터박스 처리

### 2. 파일 관리
- **지원 포맷**: MP4, MOV, M4V, HEVC, H.264
- **파일 소스**: 
  - Files 앱 (기기 내부 저장소)
  - iCloud Drive
- **파일 추가**: Drag & Drop으로 여러 파일 동시 추가
- **재생 목록**: 추가한 순서대로 자동 정렬

### 3. 재생 컨트롤
- 재생/일시정지
- 볼륨 조정 (시스템 볼륨과 연동)
- 자동 재생 on/off (다음 영상 자동 재생)
- 길게 눌러서 빠른 탐색 (2x, 4x 속도)
- 재생 진행바 (seek bar)
- 이전/다음 영상 이동

### 4. 추가 고려사항
- **제스처 지원**:
  - 두 손가락 핀치로 화면 비율 조정
  - 좌우 스와이프로 10초 앞/뒤 이동
- **백그라운드 재생**: PiP(Picture in Picture) 지원
- **재생 기록**: 마지막 재생 위치 저장
- **플레이리스트**: 재생 목록 저장/불러오기

## 기술 스택 제안
- **언어**: Swift
- **UI 프레임워크**: SwiftUI
- **비디오 재생**: AVKit, AVFoundation
- **파일 접근**: DocumentPicker, FileManager

## 개발 우선순위
### 1단계 (2-3주)
- 기본 비디오 재생 기능
- Drag & Drop 파일 추가
- 일반 모드 UI 구현

### 2단계 (2-3주)
- 숏폼 모드 UI 구현
- 제스처 컨트롤
- 자동 재생 기능

### 3단계 (1-2주)
- macOS 호환성
- PiP 지원
- 성능 최적화
